本文参考自 MDN 的文档与日常使用所得，如果你对这个 API 已经相当熟悉，请帮我看下是否有遗漏（笑）；不了解的同学可以浏览一下，应该会对你有所帮助。

小结

至此，关于 toLocaleString 的介绍就暂告一段落了。可以看到，在某些场景下需要格式化对象成字符串时，可以发挥极大的作用，不再需要自己苦兮兮地写函数做转换。尽管这个 API 稍微冷门一点，但还是很有意思的。
